## Description

CRM tasks api

## Project setup

```bash
$ npm install
```

## Open
http://locahost:3000 - api
http://locahost:3000/swagger - swagger

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
